# 🎮 Gaming Store

متجر إلكتروني لشحن الألعاب بدعم كامل للعربية والإنجليزية

## ✨ المميزات

- ✅ واجهة حديثة وسريعة
- 🌐 دعم اللغة العربية والإنجليزية
- 🛒 نظام سلة التسوق الكامل
- 💳 نموذج طلب وسداد آمن
- 🎮 دعم ألعاب متعددة (Free Fire, Clash of Clans, eFootball)
- 📱 تصميم متجاوب (Responsive Design)

## 🚀 البدء السريع

### المتطلبات
- Node.js (v14+)
- npm أو yarn

### التثبيت المحلي

```bash
# انسخ المستودع
git clone https://github.com/mama900ai-wq/gaming-store.git
cd gaming-store

# ثبت المكتبات
npm install

# شغل المشروع محلياً
npm run dev

# افتح المتصفح على
# http://localhost:3000
```

## 🌐 الرفع على Vercel

### الطريقة 1: من خلال الواجهة (الأسهل)

1. اذهب إلى [Vercel.com](https://vercel.com)
2. سجل دخول باستخدام حسابك على GitHub
3. اضغط على **New Project**
4. اختر مستودعك `gaming-store`
5. اضغط **Deploy**
6. انتظر قليلاً... تم! 🎉

### الطريقة 2: من خلال CLI

```bash
# ثبت Vercel CLI
npm install -g vercel

# ادخل إلى مجلد المشروع
cd gaming-store

# شغل أمر الرفع
vercel

# اتبع التعليمات
```
